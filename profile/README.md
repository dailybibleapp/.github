## What you'll find here

- **Well-structured repositories** following shared engineering standards.
- **Open collaboration** through issues, pull requests, and discussions.
- **Security-first defaults** — see our [Security Policy](../SECURITY.md).

## Getting started

| I want to…                  | Go to                                              |
| --------------------------- | -------------------------------------------------- |
| Report a bug                | Open an issue using the **Bug report** template    |
| Request a feature           | Open an issue using the **Feature request** template |
| Contribute code             | Read the [Contributing Guide](../CONTRIBUTING.md)  |
| Report a vulnerability      | Follow the [Security Policy](../SECURITY.md)       |
| Get help                    | See [Support](../SUPPORT.md)                       |

## Our standards

All contributors follow our [Code of Conduct](../CODE_OF_CONDUCT.md) and
[Contributing Guide](../CONTRIBUTING.md). Technical decisions are guided by
maintainability, security, performance, and clarity.
