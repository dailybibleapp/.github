# Support

Looking for help? Here's where to go.

## Questions and discussions

- **General questions / ideas:** open a thread in the relevant repository's
  Discussions (or the organization Discussions, if enabled).
- **Documentation:** check the `README.md` of the specific repository first.

## Reporting problems

- **Bugs:** open an issue using the **🐞 Bug report** template.
- **Feature requests:** open an issue using the **✨ Feature request** template.

## Security issues

**Do not** report security vulnerabilities through public issues or discussions.
Follow the process in our [Security Policy](SECURITY.md).

## What to include

When asking for help, include enough context to reproduce or understand the
issue — but **never include sensitive data** (credentials, personal data, or
internal details). Redact logs and screenshots first.
